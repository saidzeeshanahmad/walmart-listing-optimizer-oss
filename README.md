# Walmart Listing Optimizer OSS (WLO-OSS)

An open-source toolkit designed to help small and medium eCommerce sellers optimize Walmart product listings using AI-assisted workflows.

## 🚀 Features
- AI-powered title generation
- Keyword extraction for better search ranking
- Product description optimization
- Bulk listing formatting support
- - Simple CLI-based workflow for quick usage

## 🎯 Purpose
Many small sellers cannot afford expensive SaaS tools. This project provides a free, open-source alternative to improve listing quality and visibility.

## 🛠️ Installation
```bash
git clone https://github.com/saidzeeshanahmad/walmart-listing-optimizer-oss.git
cd walmart-listing-optimizer-oss
pip install -r requirements.txt
```

## ▶️ Usage
```bash
python src/title_generator.py --input examples/sample_input.json
```
